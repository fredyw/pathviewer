# pathviewer
A very tiny tool to pretty-print path list from an environment variable.

# How to build
`go build pathviewer.go`

# Usage
`pathviewer <env_variable>`

# Example
    pathviewer PATH
    
    PATH:
    - /usr/local/sbin
    - /usr/local/bin
    - /usr/sbin
    - /usr/bin
    - /sbin
    - /bin
    - /usr/games
    - /usr/local/games
